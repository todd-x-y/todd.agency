# todd.agency

Professional site for Todd Youngblood's data engineering and information systems design practice.

## About

This site serves as a landing page for consulting engagements focused on rapid prototyping, systems design, and making complex operations visible and actionable for organizations.

## Stack

- React with Tailwind CSS
- Static hosting via GitHub Pages

## Local Development

Open `index.html` directly in a browser, or serve locally:

```
npx serve .
```

## Deployment

Pushes to `main` are published automatically via GitHub Pages.
